# 📌Didabara

## **기업,동아리 또는 스터디 그룹을 위한 업무/정보 제공 및 운영 관리 서비스**

### **초대코드를 통해 커뮤니티를 구독**

### **구독시 해당 커뮤니티에 업로드된 정보 제공**

### **구독 리스트를 통해 내가 참여하고 있는 커뮤니티 관리**

### **ppt, excel, docx, jpg 등 의 파일 업로드, 미리보기,다운로드 기능**

### **같은 커뮤니티를 구독 하고 있는 사용자들만의 채팅,댓글 기능**
